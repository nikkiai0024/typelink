# CLAUDE.md - MBTI TypeLink

## アプリ概要
**TypeLink** - クイックMBTI診断 × シェアカード × 友達相性

「96問もある診断、やる気でない。結果をXにシェアしたい。友達と比べたい。」を全部解決する。

## 解決する問題（競合の弱点）

1. **既存最強アプリ（MBTIテスト 4.5★ 5,808件）** の不満
   - 96問で途中離脱が多い → クイック20問モードで解決
   - 結果をSNSにシェアするための画像がない → シェアカード自動生成
   - 友達との相性を確認する場がない → 友達相性診断

2. **日本市場特有の需要**
   - K-POPカルチャーの影響でMBTIブーム継続中（特にZ世代）
   - TikTok/X で「#MBTI」「#INFJ」コンテンツが大量流通
   - 診断結果カードをシェアする文化が定着

## 主要機能（MVP）

### 1. クイック診断（20問）
- 標準的なMBTI質問セット（I/E、N/S、T/F、J/P の各5問ずつ）
- 全部答えたら即結果表示
- ログイン・登録不要

### 2. 診断結果画面
- 16タイプのうちどれか表示
- タイプの説明文（日本語）
- 有名人の例（INFJならルイ・ヴィトン創始者など）
- 強み・弱み・恋愛傾向

### 3. シェアカード生成（★独自機能）
- 結果をおしゃれなカード画像として生成（UIImage として）
- タイプ名・アイコン・キャッチコピーをデザインした画像
- 「シェア」ボタンで iOS 標準シェアシートを開く
- X（Twitter）、Instagram、LINE でシェア可能

### 4. 友達相性診断（★独自機能）
- 「友達のタイプを選ぶ」ページ
- 自分のタイプ × 友達のタイプ → 相性スコア + コメント（例：「最高の組み合わせ！」）
- 16×16 = 256通りの相性テキストをハードコード（APIなし）

### 5. 全タイプ一覧
- 16タイプのカード一覧
- タップで詳細を見る

## 技術スタック

- **React Native + Expo** (TypeScript)
- **ローカルデータのみ** - APIサーバー不要、診断ロジックもデータもすべてアプリ内
- **expo-sharing** - シェアカード用
- **react-native-view-shot** または Canvas API - カード画像生成

## 画面一覧

1. **ホーム画面** - 「診断スタート」ボタン + 全タイプ一覧へのリンク
2. **診断画面** - 1問ずつ表示（プログレスバーつき）
3. **結果画面** - タイプ名・説明・シェアボタン・相性チェックへのリンク
4. **相性診断画面** - 16タイプから友達のタイプを選択
5. **相性結果画面** - スコアとコメント
6. **タイプ詳細画面** - 各タイプの詳細情報

## デザイン方針

- **カラー**: メインカラー `#7B5EA7`（紫）+ 背景 `#F7F5FB`（淡ラベンダー）
- **トーン**: ミニマル + おしゃれ。韓国系UIの影響を受けた清潔感
- **フォント**: システムフォント（-apple-system）でネイティブ感
- **カード**: 角丸・シャドウ・グラデーション活用
- **シェア画像**: 1080×1080px、タイプ別カラーバリエーション

## データ設計（ハードコード）

### 診断質問（20問）
各軸5問 × 4軸 = 20問

### タイプデータ（16件）
```json
{
  "INFJ": {
    "name": "提唱者",
    "description": "...",
    "strengths": ["共感力が高い", "理想主義"],
    "weaknesses": ["燃え尽きやすい", "完璧主義"],
    "color": "#4A90E2",
    "emoji": "🌟",
    "shareCaption": "私はINFJ。静かな理想主義者。"
  }
}
```

### 相性データ（256通り）
```json
{
  "INFJ_ENTP": {
    "score": 95,
    "label": "最強コンビ",
    "comment": "お互いの弱点を補い合える黄金コンビ。"
  }
}
```

## 優先実装順

6. UI ポリッシュ

## 注意事項

- **MBTI自体の質問は著作権に注意** → 「MBTI的な質問」として独自表現で書く（診断テスト内容はMyers & Briggsの商標だが、類似の質問を独自表現で作るのは一般的に合法）
- シェアカード内に「MBTIテスト」という商標表現を入れない → 「性格タイプ診断」と表記
- 医療・心理診断として表記しない（Disclaimer追加）

## ターゲットユーザー

- 10〜20代の日本人（特に女性）
- K-POP・韓国カルチャー好き
- X/Instagramでのシェアが好き
- 友達との会話のネタにしたい
